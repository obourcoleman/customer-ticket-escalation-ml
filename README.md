# Customer Support Ticket Escalation Prediction

## Project Overview
This project uses machine learning to predict whether a customer support ticket will be escalated.

## Dataset
- Source: Kaggle  
- Link: https://www.kaggle.com/datasets/mirzayasirabdullah07/customer-support-tickets-dataset-200k-records  

## Objective
To classify tickets into escalated or not escalated.

## Features Used
- Text: issue_description (TF-IDF)
- Categorical features
- Numerical features

## Models Used
- Logistic Regression
- Random Forest

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC

## Results
The model achieved around 50% accuracy, indicating that escalation prediction is a challenging task.
